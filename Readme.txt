# Digital Marketing Course Platform

This project is a customized course enrollment and portfolio website based on the EasyFolio template.

## Features

- Digital marketing course catalog (beginner, intermediate, advanced)
- Course details with price and discount display
- Enroll process: Sign Up / Sign In, then payment/contact form
- Responsive design using Bootstrap 5
- Consistent navigation menu (Home, Service, Sign Up/Sign In, Payment, Contact, About)
- Contact and support information

## How to Use

1. Browse courses on the Home or Service page.
2. Click "Enroll Now" to view course details, price, and discounts.
3. Proceed to Sign Up / Sign In, then fill the payment/contact form.
4. Payment instructions will be sent to your email.
